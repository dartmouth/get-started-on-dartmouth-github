Add Dartmouth Email

Now that you have a GitHub account, it is time to add your Dartmouth email to it in two forms

  • name@dartmouth.edu (might already be there if you created the account for work only)
  • netid@dartmouth.edu
Note: that each address is needed and serves different purposes. The name@dartmouth.edu will be linked to your Git commit history and netid@dartmouth.edu ensures Dartmouth Single Sign-On (SSO) works despite name/email changes.

Steps

  • Log into GitHub
  • Browse to settings/emails (in the top right click your profile icon and then "Settings", then on the left click "Emails")
  • Under the "Add email address" section, add both
    • name@dartmouth.edu
    • netid@dartmouth.edu
  • Go to your mailbox and click the link to complete verification for both addresses

That's it. You now have a fully setup GitHub account and you're ready to be added as a member of one or more Dartmouth GitHub organizations.

Next steps

If you wish to join https://github.com/dartmouth, please browse to https://dashboard.dartmouth.edu/github/invitation_request/ and use the "Request Invitation" button.

Access to other Dartmouth's GitHub organizations is limited to invite only. If you were sent these instructions from someone you should also expect them to invite you to the appropriate organizations or give you further instruction.

Return home.
